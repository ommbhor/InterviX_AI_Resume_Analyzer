# 🚀 AI Resume Analyzer & Interview Generator

A full-stack AI-powered web application that simulates a real-world hiring assistant.

Users can upload resumes, analyze job descriptions, detect skill gaps, and generate AI-powered interview questions along with ATS-optimized resumes.

---

## 📌 Features

- 📄 Resume Upload & Parsing
- 🧠 AI-Based Skill Extraction
- 📊 Skill Gap Detection (Resume vs Job Description)
- 🤖 AI Interview Question Generator
- 📑 ATS-Optimized Resume Generation
- 🔐 Secure Authentication (JWT + Token Blacklisting)
- 📁 Resume PDF Generation using Puppeteer
- 📈 Interview Report Management System
